Roundabout_XH - ein CMSimple_XH Plugin

Roundabout_XH (alias "Der mit der Galerie tanzt") zeigt Bildergalerien als Karussell. Dazu wird ausschließlich JavaScript verwendet – kein Flash- oder ähnliches Plugin wird benötigt. Roundabout_XH hat keine intergrierte Galerieverwaltung; statt dessen werden Galerien verwendet, die mit KissGallery_XH oder FotoGalerie_XH erstellt wurden.

Voraussetzungen

Roundabout_XH ist ein Plugin für CMSimple_XH. Es benötigt eine UTF-8 kodierte Version und jQuery4CMSimple.

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH-Wiki finden Sie weitere Details.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Rechner.
  3. Laden Sie das ganze Verzeichnis roundabout/ auf Ihren Server in CMSimple_XHs Plugin-Verzeichnis hoch.
  4. Machen Sie die Unterverzeichnisse config/, css/ und languages/ beschreibbar.
  5. Gehen Sie zu "Roundabout" im Administrationsbereich, um zu prüfen, ob alle Voraussetzungen erfüllt sind.

Einstellungen

Die Plugin-Konfiguration erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Website. Wählen Sie "Plugins"→"Roundabout".

Sie können die Voreinstellungen von Roundabout_XH unter "Konfiguration" ändern. Hinweise zu den Optionen werden beim Überfahren der Hilfe-Icons mit der Maus angezeigt. Die wichtigsten Einstellungen sind "gallery data", "gallery images" und "gallery separator"; die Vorgaben gelten für FotoGalerie_XH. Ändern Sie diese, wenn Sie den Speicherordner der FotoGalerie_XH geändert haben, oder wenn Sie Roundabout_XH mit KissGallery_XH verwenden wollen.

Die Lokalisierung wird unter "Sprache" vorgenommen. Sie können die Sprachtexte in Ihre eigene Sprache übersetzen, oder diese Ihren Wünschen gemäß anpassen.

Das Aussehen von Roundabout_XH kann unter "Stylesheet" angepasst werden.

Verwendung

Um ein Karussell mit allen Bildern einer Galerie anzuzeigen, rufen Sie einfach das Plugin mit der Galerie ID als Parameter auf, z.B. auf einer Seite:

{{{PLUGIN:roundabout(1);}}}

Oder im Template:

<?php echo roundabout(1);?>

Das Anklicken eines der Bilder öffnet das Bild in voller Größe in einer Lightbox.

Einschränkungen

Um das Karussell sehen zu können, muss JavaScript im Browser des Besuchers aktiviert sein. Ansonsten wird eine Meldung angezeigt, die in den Spracheinstellungen festgelegt werden kann.

Die selbe Galerie darf nicht mehrmals auf der selben Seite aufgerufen werden (unabhängig davon, ob Sie aus dem Content, einer Newsbox oder dem Template aufgerufen wird).
